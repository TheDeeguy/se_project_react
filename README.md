# WTWR (What to Wear?)

## About

WTWR is a React application that helps users decide what to wear based on weather conditions. Users can view weather information and manage a collection of clothing items suitable for different temperatures.

## Features

- Weather-based clothing recommendations
- Add and delete clothing items
- Temperature unit toggle (°F/°C)
- Responsive design
- Modal interfaces for user interactions

## Technologies Used

- React
- JavaScript (ES6+)
- CSS3
- Vite
- JSON Server (for local development)
